# E-Commerce SQL Project

## Project Overview
This project is based on an E-Commerce Database using MySQL. 
It includes customer, products, orders, payments, and sales analysis.

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT, SUM, AVG
- JOIN & LEFT JOIN
- HAVING
- Subqueries

## Database Tables
- Customers
- Products
- Orders
- Order Items
- Payments

## Key Analysis
- Customer Order Analysis
- Revenue Analysis
- Payment Insights
- Product Analysis

## Tools Used
- MySQL Workbench
- SQL
